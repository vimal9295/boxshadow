# What is this?

Get Perfect Shadows For Images

#Installation

`npm i boxshadow --save`

Then..

...

import { boxshadow } from 'boxshadow';

boxshadow({
	box_type:'soft',
	padding:false
});
.....

## Options

Box Shadow Supports 2 options,
* *box_type* - hard | soft(default)
* *padding* - _boolean_ (Defaults to false)
